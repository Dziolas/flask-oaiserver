Installation
============

Install Flask-OAIServer with ``pip`` ::

    pip install flask-oaiserver

The development version can be downloaded from `its page at GitHub
<http://github.com/inveniosoftware/flask-oaiserver>`_. ::

    git clone https://github.com/inveniosoftware/flask-oaiserver.git
    cd flask-oaiserver
    python setup.py develop
    source run-tests.sh

Requirements
------------
Flask-OAIServer has the following dependencies:

* `Flask <https://pypi.python.org/pypi/Flask>`_
* `six <https://pypi.python.org/pypi/six>`_

Flask-OAIServer requires Python version 2.6, 2.7 or 3.3+
